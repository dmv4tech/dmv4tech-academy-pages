Scratch Programming

Age Group: 6–10 years

Duration: 8 weeks (1 session/week)

Level: Beginner

What Kids Will Learn

  • Understanding coding logic with drag-and-drop blocks
  • Creating fun animations and interactive stories
  • Designing basic games with characters and sounds
  • Problem-solving through play

Tools Used

Scratch (MIT), Internet browser (no installation required)
